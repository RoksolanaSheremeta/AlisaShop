= Litesite Responsive Wordpress Theme =

Litesite Wordpress theme for company websites. Litesite is a responsive business Minimal setting theme which
can be used for blog or Corporate website. It supports custom page templates, custom widgets, custom logo, 
drop down menu, threaded comments. It is simple, clean and responsive theme. 
You can add your own logo from the wordpress admin.
For questions, comments or bug reports, please go to WordPress forums (http://wordpress.org/support/).

/*========= Theme Features =========*/

* Custom Home Page
* Simple, Clean and Responsive
* Sidebar, Footer
* Three page layouts
* Font Awesome included
* Add your own logo from the wordpress Admin under Appearance -> Customize
* Drop-down Menu
* Cross-browser compatibility
* Threaded Comments

/*========= Installation =========*/

You can install the theme through the WordPress installer under "Themes" > "Install themes" by searching for it.
Alternatively you can download the file, unzip it and move the unzipped contents to the "wp-content/themes" folder
of your WordPress installation. You will then be able to activate the theme.

/*========= How to =========*/

1) How to add social icons in sidebars ?
--> Litesite come with inbuilt social widget. To add social icons to your sidebar follow the blow steps
	Go under 'Appearance' -> 'widgets' in WordPress Admin and drag 'litesite social widget'
	Enter your social profile urls save and you are ready to go.
	
2) How to create Home page ?
--> Litesite come with Home page template which can be used for a separate Home Page and display sticky posts as featured
    posts etc. Go to Settings-> Reading in WordPress admin select static page and select the page which you want
    to use as Home page from the dropdown.
    Your recent sticky post will be display on the home pages as featured posts.
